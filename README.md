# Mask Dedection

## Install / Setup Package

```
pip install - r requirements.txt
```

## Packages Used
  - Tensorflow
  - Keras
  - Numpy
  - Pandas
  - Opencv
  - Imutils
  - os
  - Sklearn
  - Matplotlib

## Started

```
python dedect-mask-video.py
```
